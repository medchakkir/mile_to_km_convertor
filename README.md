# Mile to Kilometer Converter

A simple desktop application that converts distances from miles to kilometers. This user-friendly tool provides quick and accurate conversions with a clean, intuitive interface.

## Features

- Real-time conversion from miles to kilometers
- Clean and minimalist user interface
- Precise calculations with rounded results
- Input validation for numerical values
- Instant conversion with a single click

## Requirements

- Python 3.x
- Tkinter (built into Python standard library)

## Installation

1. Clone this repository:

```bash
git clone https://github.com/momed-ali01/mile_to_km_convertor.git
cd mile_to_km_convertor
```

## Usage

Run the application:

```bash
python main.py
```

### How to Use

1. Enter the number of miles in the input field
2. Click the "Calculate" button
3. The converted value in kilometers will be displayed below

### Conversion Formula

The application uses the following conversion formula:

- 1 mile = 1.689 kilometers
- Results are rounded to the nearest whole number

## Interface Elements

- Input Field: Enter the number of miles
- Calculate Button: Triggers the conversion
- Result Display: Shows the converted value in kilometers
- Clear Labels: Indicates input and output units
